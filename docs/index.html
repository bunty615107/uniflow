⚙️🌊 UniFlow

Secure, connection-agnostic Managed File Transfer (MFT) orchestrated with Rust, Tokio, and Rayon.

View on GitHub

Connection-Agnostic

Jobs are orchestrated independently of their source or destination, supporting robust multi-path transfers and failover.

High Performance

Built with Rust, leveraging Tokio for asynchronous I/O and Rayon for parallel computation and delta hashing.

Built-in Security

Secure by design with credential vaults, block signatures, and integrated multi-surface multi-tenant design.

Clean Architecture

Strictly layered into domain, application, and infrastructure layers for maintainability and extensive extensibility.
